## Requirements 📦

- Flask

## Installation ⬇️

download source code from Github 💾
`git clone https://github.com/Muntasir101/Simple_Weather.git`

go to directory 📁
`cd Simple_Weather`

install requirements.txt 🔽
`pip install -r requirements.txt`

it's ready to run 🎉
`python app.py`
